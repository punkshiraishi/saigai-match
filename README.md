# サイガイマッチとは

災害時の救援物資プラットフォームを意識して作成中のアプリケーションです。


# URL

[サイガイマッチ](https://saigai-match.herokuapp.com/)


# 運用方法

以下のようにgithub-flowで運用する。
1. 開発用ブランチを切って作業
2. 開発用ブランチへ commit & push
3. master への pull request を作成
4. 管理者が master へ merge した後本番デプロイ
